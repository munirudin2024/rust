Panduan Pemakaian Project Rust (data_cleaner)

Lokasi project:
- source utama ada di folder: ds/

1) Masuk ke folder project
cd ds

2) Build project
cargo build

3) Jalankan untuk 1 file CSV
cargo run -- data/dataset_transaksi_ecommerce.csv

4) Jalankan untuk banyak file CSV sekaligus
cargo run -- data/file1.csv data/file2.csv

5) Aktifkan mode hard reject (karantina ketat)
cargo run -- data/dataset_transaksi_ecommerce.csv --hard-reject

6) Atur cutoff tanggal bisnis manual
Format tanggal: YYYY-MM-DD
cargo run -- data/dataset_transaksi_ecommerce.csv --max-date 2026-04-17

Catatan penting argumen:
- Minimal harus ada 1 file .csv
- Argumen yang didukung:
  --hard-reject
  --max-date YYYY-MM-DD

Output yang dihasilkan (di dalam ds/output/):
- output/csv/                       -> hasil CSV utama
- output/csv/transaksi_clean/       -> tabel payload bersih
- output/csv/transaksi_audit_log/   -> tabel audit log
- output/csv/kpi_kualitas/          -> KPI kualitas data
- output/csv/per_tahun/<tahun>/     -> split data per tahun
- output/quarantine/                -> hasil karantina dan investigasi
- output/laporan/                   -> presentasi txt/html
- output/html/report_data.json      -> ringkasan JSON lintas dataset

Contoh alur cepat:
cd ds
cargo build
cargo run -- data/dataset_transaksi_ecommerce.csv --hard-reject

Setelah selesai, cek ringkasan path output yang dicetak di terminal.
